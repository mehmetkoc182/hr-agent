# Recruiter AI Agent

This is a simple job recruiter agent that finds suitable jobs for job seekers.

# Quickstart Guide:

Install dependencies
`npm install`

Run the service "Default at port 3000"
`npm run dev`

Tools Used:
- External API for job searches (https://jobicy.com/)
- Calculator tool
- Resume upload tool
